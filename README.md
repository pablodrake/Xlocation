# X Country

A Chrome extension that displays the account location (country flag) for users on X (formerly Twitter).

## Features

-   **Timeline Support**: Shows country flags next to usernames in the timeline.
-   **DM Support**: Displays country flags in Direct Messages.
-   **Auto-Fetch**: Automatically fetches country data when flags enter the viewport.
-   **Smart Caching**: Caches fetched locations to minimize API requests.
-   **Algorithmic Flag Generation**: Generates flag emojis from ISO 3166-1 alpha-2 country codes.

## Installation

1.  Clone this repository.
2.  Open Chrome and go to `chrome://extensions`.
3.  Enable "Developer mode" in the top right.
4.  Click "Load unpacked" and select the extension directory.
